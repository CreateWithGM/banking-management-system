Savings Tracker Feature Description

Initial Deposit and Goal Amount:

  • Users can set their initial deposit amount within the range of ₹20,000 to ₹50,000.
  • The goal amount is user-defined but must be a minimum of ₹10,000. Users can choose any amount above this minimum.

Selected Days and Transactions:

  • Users are allowed to select up to 10 days to use their savings within the tracker.
  • Each day allows for a maximum of 1 transaction, ensuring controlled usage and planning.

Balanced Transactions and Interest Calculation:

  • Transactions must be balanced equally throughout the selected days to avoid imbalanced usage patterns.
  • If transactions are balanced, a 2% interest is added to the savings account along with the daily average transaction amount.
  • The daily average transaction amount is calculated based on the total transactions made during the goal period divided by the number of days.

User Notification and Information:

  • Users are not informed about these criteria explicitly by the system.
  • The system silently checks for balanced transactions (deviation within 10% of the average transaction) without user notification.

Interest and Average Transaction Amount Addition:

  • If transactions are balanced and the goal is achieved, the system adds a 2% interest along with the daily average transaction amount to the savings account at the end of the goal period.
  • This interest and average transaction amount are added to the savings account once the goal is achieved and transactions are balanced.

Overall Functionality:

  • The Savings Tracker feature aims to encourage disciplined and balanced usage of savings over a limited period, rewarding users with interest and average transaction amount additions if they meet the balanced transaction criteria.
  • Users can track their savings progress without explicit notifications, relying on the system's monitoring and message display based on transaction balance checks.
